---

## What This Repository Contains

- Interactive HTML player radar cards
- An indexed homepage linking all players
- A clean, mobile-friendly viewing experience
- Static-ready visuals designed for social sharing
- Public hosting via GitHub Pages

---

## What This Repository Does *Not* Contain

- Source code for data collection
- Source code for analytics or scoring logic
- Automation scripts or build pipelines
- Proprietary metric definitions

The visuals published here are **outputs**, not raw methodology.

---

## Project Overview

The player value radar cards are designed to present complex basketball data in a format that is:

- Easy to interpret
- Visually engaging
- Interactive on desktop and mobile
- Suitable for both casual fans and analytically minded viewers

Each radar represents a consistent analytical framework applied across players to support comparison and contextual understanding.
